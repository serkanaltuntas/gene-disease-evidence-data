# Gene-Disease Evidence Index reproducibility subset

Release: `top120-public-preview-v0.1.3`

This directory is a public, leak-audited reproducibility subset for the
Gene-Disease Evidence Index public preview. It supports inspection of release
checksums, manuscript-facing table snapshots, figure inputs, static SVG figures,
and public-only table regeneration from the already-public release artifacts.

It is not the full production pipeline. The production ingestion, release-gate,
private review, and manuscript-development repositories remain outside this
public subset.

## Public-only rebuild

The helper script below reads only public repository files:

- `releases/top120-public-preview-v0.1.3/batch-manifest.json`
- `releases/top120-public-preview-v0.1.3/integrity.json`
- child release directories under `releases/*-top120-v0.1.1-preview/`
- `archives/gene-disease-evidence-top120-public-preview-v0.1.3.zip`

Run from the repository root:

```bash
python3 docs/reproducibility/top120-public-preview-v0.1.3/scripts/rebuild_public_methods.py
```

By default, the script writes regenerated public-only tables to
`docs/reproducibility/top120-public-preview-v0.1.3/generated/public-release/`
and compares them with `tables/public-release/`.

The public-only rebuild covers:

- dataset overview;
- disease release counts;
- source support, silence, and conflict counts;
- disease-level source coverage;
- source freshness as represented in public source metadata;
- support-state counts;
- Open Targets datatype and datasource channel summaries from public
  `evidence_rows.jsonl` files.

Regenerate figure-input JSON snapshots from the subset table snapshots:

```bash
python3 docs/reproducibility/top120-public-preview-v0.1.3/scripts/rebuild_figure_inputs.py
```

The figure-input helper uses the public-release tables plus the included
review-gate snapshot table. It does not import or execute the private pipeline.

## Snapshot tables and figures

`tables/public-release/` contains table snapshots that should be reproducible
from public release artifacts with the helper script.

`tables/review-gate-snapshots/` contains manuscript-supporting review-gate table
snapshots. These are included to make the manuscript support layer visible, but
they are not regenerated by the public-only helper because their source
artifacts are private release-gate records.

`tables/manuscript-support/` contains the simulated manuscript candidate
contract table. This table is a manuscript-supporting methods artifact; it does
not change the fixed public preview and is not a release-candidate
recommendation.

`figures/inputs/` and `figures/svg/` contain generated figure-input and SVG
snapshots for the current manuscript-supporting display set.

## Scope boundary

The public preview is a research-use, source-attributed candidate association
resource. It is not medical advice, diagnosis, treatment guidance, clinical
decision support, patient risk prediction, clinical certainty, clinical
validity, clinical utility, external peer review, or clinical expert curation.

Source silence means that an included source did not contribute supporting
evidence for a release-scoped candidate association. It is not evidence of
biological absence, clinical absence, or evidence absence outside the included
source set.
