# alzheimer-top120-v0.1.1-preview

Public preview data package for research-use evaluation of source-attributed
gene-disease evidence records.

This release is not a public scientific finding and is not manuscript-grade. It exists to
make the data package, provenance, source boundaries, and review gates inspectable before
any paper-grade release.

Do not use this package as medical advice, diagnosis, treatment guidance, patient risk
prediction, clinical decision support, clinical certainty, or a clinical-use dataset.
