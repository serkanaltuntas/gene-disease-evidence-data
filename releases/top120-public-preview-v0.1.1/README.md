# Gene-Disease Evidence Index 120-Association Public Preview v0.1.1

This aggregate release packages the twelve disease-scoped 120-association public preview releases for research-use inspection. It is a public data package with paper-grade gate artifacts passed in the private pipeline, but it is not clinical guidance, medical advice, diagnosis, treatment guidance, patient risk prediction, clinical decision support, or clinical certainty.

## Contents

- `batch-manifest.json` lists the twelve disease release directories and aggregate counts.
- `integrity.json` records SHA-256 checksums for the aggregate release files and downloadable archive.
- `archives/gene-disease-evidence-top120-public-preview-v0.1.1.zip` is the deterministic aggregate download bundle.

## Scope

The release contains source-attributed candidate gene-disease evidence records. Source silence is provenance context only and is not evidence of absence. Advisory referee review and sampled human review are release gates; they are not external peer review or clinical expert curation.
