# Gene-Disease Evidence Index top120-public-preview-v0.1.3

This is the current 12-disease, 120-association public preview aggregate for the
Gene-Disease Evidence Index. It is a research-use package of source-attributed
candidate gene-disease associations, provenance records, public integrity
metadata, and review-gate summaries.

This patch release keeps candidate association content and archive layout stable
relative to `top120-public-preview-v0.1.2`. It updates release citation metadata
so the repository `CITATION.cff`, aggregate release id, archive name, and
integrity manifest all refer to `top120-public-preview-v0.1.3`.

## Scope

- Diseases: 12
- Associations: 1,440
- Unique genes: 1,138
- Evidence rows: 1,505
- Source snapshots: 24
- Advisory review artifacts: 5,760

## Boundary

These artifacts are research-use resources. They are not medical advice,
diagnostic guidance, treatment guidance, patient data, patient risk prediction,
clinical decision support, clinical certainty, clinical validity, or clinical
utility material. Paper-grade gates passed in the private pipeline, but that does
not imply manuscript acceptance, external peer review, or clinical validity.
